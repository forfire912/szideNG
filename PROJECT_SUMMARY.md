## 🚀 如何使用

### 快速开始

1. **准备工具链**
   ```bash
   # 下载必要组件
   - VSCodium 1.85.0+
   - ARM GCC 12.3+ (用于 STM32)
   - ADI CCES 2.11+ (用于 ADSP)
   ```

2. **解压到指定位置**
   ```
   szideNG/
   ├── vscodium/       # 解压 VSCodium
   ├── toolchains/
   │   ├── arm-gcc/    # 解压 ARM GCC
   │   └── adi-cces/   # 安装 CCES
   ```

3. **运行打包**
   ```batch
   scripts\package.bat
   ```

4. **安装使用**
   ```batch
   cd dist\szideNG-v1.0.0
   setup.bat
   ```

5. **验证**
   ```batch
   validate.bat
   ```
